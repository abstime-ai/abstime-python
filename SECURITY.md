# Security Policy

If you believe you have found a security issue in this library, please do not report it through public GitHub issues or pull requests.

Instead, email:

- `feedback@abstime.ai`

Include as much detail as possible so the team can reproduce and investigate the issue quickly.

Typical useful details:

- affected package version
- environment details
- reproduction steps
- impact summary
- any relevant request or response metadata

AbsTime will review security reports privately and follow up through email.
